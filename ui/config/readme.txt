default.json
------------
The default configuration of the buttons used by buttons.html
This file is used if no user.json exists

user.json
---------
The last version of button config saved by the user and consumed by buttons.html

static.js
---------
Animation names used when programming buttons
Segment layout and names used when creating a lighting scene
